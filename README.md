📊 Mobile Sales Dashboard (Power BI)
📌 Overview
Interactive sales dashboard created using Power BI
Focus on mobile sales performance analysis
Helps in data-driven decision making

1🚀 Features
Monthly sales tracking
City-wise sales visualization (Map)
Total quantity trend by month
Payment method analysis (UPI, Cash, Cards)
Customer ratings breakdown
Brand & mobile model performance
KPI cards (Sales, Quantity, Transactions)
Interactive filters (Month, Brand, Payment, Day)

2🛠️ Tools & Technologies
Power BI Desktop
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization

3📂 Dataset
Mobile sales data
City-wise transactions
Payment methods
Customer ratings
Product details (Brand & Model)

4📊 Insights
Top performing cities identified
Monthly sales trends analyzed
Brand comparison done
Customer satisfaction evaluated
Payment preferences observed

5⚙️ How to Use
Download .pbix file
Open in Power BI Desktop
Use filters for analysis
Explore visuals

6📌 Key Learnings
Data cleaning
Data modeling
DAX measures creation
Dashboard design
